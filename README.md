# fdb-post-reservation

## Prerequisites

- Create a pip virtual environment (e.g. called `venv`)
    ```
    python3 -m venv venv
    ```
- Activate it
    ```
    source venv/bin/activate
    ```
- Install requirements
    ```
    pip install -r requirements.txt
    ```

## Run

```
python3 app.py
```
